## Setup instructions

To setup this project open a terminal window in the root directory.

1. Enter commands to download all dependencies:
    - composer install
    - npm install

2. Rename the file ".env.example" to just ".env" Use these commands to clear the cache.
    - php artisan config:clear
    - php artisan cache:clear

3. Generate the node files with this command:
    - npm run build && npm run dev

4. Enter command to start server:
    - php artisan serve

The project should be set up at this point.
